gimp 3.0.4 -> full installation

=========================

theme for look like photoshop: https://github.com/Diolinux/PhotoGIMP

=========================
- open gimp -> Edit (menu bar) -> Preferences -> Folders -> Plug-ins
- create folder if doesn't exist in C:\Users\{USER}\AppData\Roaming\GIMP\3.0\plug-ins
- copy&paste file plugin with ext ".py"

- open gimp -> Edit (menu bar) -> Preferences -> Folders -> Scripts
- create folder if doesn't exist in C:\Users\{USER}\AppData\Roaming\GIMP\3.0\scripts

# plugis:

- gmic => Ratusan filter dan efek kreatif untuk retouch, artistik, koreksi warna, dll.
extract file gmic_3.6.0_gimp3.0_win64.zip -> copy&paste folder gmic_gimp_qt to "C:\Users\{USER}\AppData\Roaming\GIMP\3.0\plug-ins"

- Resynthesizer => Content-aware fill / remove object selection
https://www.youtube.com/watch?v=FCi_S14jBYM
https://gimpchat.com/viewtopic.php?f=7&t=21535#p294979
extract file "Resynthesizer3.0.zip" =>
  -> copy&paste all folder in "plug-ins" to "C:\Users\{USER}\AppData\Roaming\GIMP\3.0\plug-ins"
