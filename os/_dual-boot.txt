## not entry menu grub

1) sudo vim /etc/default/grub
GRUB_TIMEOUT=10
GRUB_TIMEOUT_STYLE=menu
#GRUB_HIDDEN_TIMEOUT
2) sudo update-grub
3) restart laptop

## windows menu grup not found

1) sudo vim /etc/default/grub
GRUB_DISABLE_OS_PROBER=false
2) sudo os-prober
3) sudo update-grub
4) restart laptop

## (MBR) remove dual boot, just windows!

1) Insert os windows
2) Open app "Create and format hard disk partitions" and delete valume linux (Unallocated)
3) open cmd (administrator) and run:
bootsect /nt60 C: /mbr
4) restart laptop
5) Open app "Create and format hard disk partitions", right click partition C, select "Extend Volume", ikuti wizard sampai selesai (ambil semua ruang Unallocated).

## (GPT) remove dual boot, just windows!

1) Insert os windows
2) run "win + r" then type "msconfig", select tab 'boot' then delete os linux and click "apply"
3) Open app "Create and format hard disk partitions" and delete valume linux (Unallocated)
4.1) Open app "Create and format hard disk partitions"
4.2) select volume 'efi' and right click then select "Change Drive Letter and Paths"
4.3) click 'Add' and select 'S' (example) then 'Ok'
4.4) open 'Windows Explorer' entry drive 'S:' 
4.5) delete folder linux (don't delete folder Microsoft & Boot)
4.6) close 'Windows Explorer'
4.7) Open app "Create and format hard disk partitions" again
4.8) select volume 'efi' 'S:' and right click then select "Change Drive Letter and Paths" again
4.9) click 'Remove' then 'Ok'
5) restart laptop
6) Open app "Create and format hard disk partitions", right click partition C, select "Extend Volume", ikuti wizard sampai selesai (ambil semua ruang Unallocated).
